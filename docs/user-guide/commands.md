# Команды CLI

## Общий синтаксис

```bash
xssguard <команда> [опции] [аргументы]
```
## Команда `scan`
Сканирует файлы и папки на наличие уязвимостей.

### Синтаксис

```bash
xssguard scan [опции] <путь>
```

### Обязательные аргументы

|Аргумент|	Описание|
|--------|----------|
|путь	 |Путь к файлу или папке для сканирования|

### Опции

|Опция|	Короткая|	Описание|	По умолчанию|
|-----|---------|-----------|---------------|
|--config|	-c	|Путь к файлу конфигурации|	-|
|--output|	-o	|Путь для сохранения отчета|	-|
|--format|	-f	|Формат отчета: console, json, html|	console|
|--verbose|	-v	|Подробный вывод|	false|
|--threads|	-t	|Количество потоков|	4|

### Примеры

```bash
# Базовое сканирование
xssguard scan .
xssguard scan index.php

# С сохранением отчета
xssguard scan ./src --output report.json --format json

# С конфигурацией
xssguard scan ./project --config custom.yml --verbose

# Быстрое сканирование больших проектов
xssguard scan ./large-project --threads 8
```

## Команда `version`
Показывает версию программы.

```bash
xssguard version
```

## Команда `init-config`
Создает пример конфигурационного файла.

```bash
xssguard init-config
# Создает файл xssguard.yml в текущей папке
```
## Коды возврата
|Код	|Значение|
|-------|--------|
|0	|Успешно, уязвимости не найдены|
|1	|Успешно, найдены уязвимости|
|2	|Ошибка в аргументах|
|3	|Ошибка сканирования|
