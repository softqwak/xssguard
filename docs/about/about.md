# О проекте XSSGuard

## Цель проекта

XSSGuard создан для автоматического обнаружения XSS-уязвимостей в исходном коде веб-приложений. 
Основная задача - помочь разработчикам находить и исправлять уязвимости на ранних этапах разработки.

## Почему XSSGuard?

- **Бесплатный и открытый** - исходный код доступен на GitHub
- **Автономный** - не требует интернета, можно использовать в закрытых сетях
- **Простой** - не нужно изучать сложный DSL, конфигурация в YAML
- **Расширяемый** - можно добавлять свои правила и языки

## Для кого этот проект

- **Web-разработчики** - для проверки своего кода
- **DevOps инженеры** - для интеграции в CI/CD
- **Студенты** - для изучения безопасности веб-приложений
- **Аудиторы безопасности** - для первичного анализа

## Технологии

- Python 3.10+
- Click - для CLI интерфейса
- Pydantic - для валидации данных
- BeautifulSoup4 - для парсинга HTML
- PyYAML - для конфигурации

## История создания

Проект начат как дипломная работа студента 4 курса. 
Цель была - создать работающий инструмент для статического анализа, 
который можно использовать в реальных проектах.
