# Архитектура XSSGuard

## Общая структура
```text
xssguard/
├── core/ # Ядро анализатора
│ ├── analyzer.py # Главный координатор
│ └── scanner.py # Управление сканированием
├── models/ # Модели данных
│ ├── vulnerability.py
│ ├── rules.py
│ └── report.py
├── plugins/ # Плагины для языков
│ ├── php/
│ ├── js/
│ └── html/
└── utils/ # Вспомогательные модули
├── file_utils.py
└── logger.py
```

## Компоненты

### 1. Ядро (Core)
Управляет процессом сканирования:
- Читает конфигурацию
- Обходит файлы
- Вызывает нужные плагины
- Собирает результаты

### 2. Модели (Models)
Классы для представления данных:
- `Vulnerability` - уязвимость
- `CodeLocation` - позиция в коде  
- `ScanReport` - отчет
- `Rule` - правило поиска

### 3. Плагины (Plugins)
Каждый плагин отвечает за один язык:
- Строит AST
- Применяет правила
- Возвращает уязвимости

### 4. Утилиты (Utils)
Вспомогательные функции:
- Работа с файлами
- Логирование
- Работа с конфигами
