# Понимание результатов

## Уровни опасности

| Уровень | Цвет | Что значит |
|---------|------|------------|
| **CRITICAL** | 🔴 Красный | Можно эксплуатировать удаленно без условий |
| **HIGH** | 🟠 Оранжевый | Высокая вероятность атаки |
| **MEDIUM** | 🟡 Желтый | Требуются дополнительные условия |
| **LOW** | 🔵 Синий | Низкий риск, сложно эксплуатировать |
| **INFO** | 🟢 Зеленый | Информационное сообщение |

## Типы уязвимостей

### Reflected XSS (Отраженный)
Данные из HTTP-запроса сразу выводятся в ответ.

**Пример:**
```php
<?php echo $_GET['search']; ?>
```
**Риск:** Злоумышленник может отправить ссылку с вредоносным кодом.

## Stored XSS (Сохраняемый)
Данные сохраняются на сервере и показываются другим пользователям.

### Пример:

```php
<?php
// Сохраняем комментарий
$comment = $_POST['comment'];
mysqli_query($db, "INSERT INTO comments SET text='$comment'");

// Позже выводим
echo $comment;
?>
```
**Риск:** Вредоносный код увидят все посетители страницы.

## DOM XSS
Уязвимость на стороне клиента в JavaScript.

### Пример:

```javascript
document.getElementById('output').innerHTML = location.hash;
```
**Риск:** Атака происходит в браузере без участия сервера.

## Форматы отчетов
### Консольный вывод (по умолчанию)
```text
🔍 XSSGuard Scan v0.1.0
📁 Path: ./project
==================================================
⚠️  УЯЗВИМОСТИ:

[HIGH] index.php:23
  → Reflected XSS
  Данные из $_GET['name'] выводятся через echo
  Исправление: используйте htmlspecialchars()

📊 Статистика:
   Файлов: 42
   Уязвимостей: 3
     HIGH: 2
     MEDIUM: 1
```
### JSON отчет
```json
{
  "scan_id": "scan-20240101-120000",
  "vulnerabilities": [
    {
      "type": "reflected_xss",
      "severity": "high",
      "location": {
        "file": "index.php",
        "line": 23,
        "content": "echo $_GET['name'];"
      }
    }
  ],
  "summary": {
    "total_files": 42,
    "total_vulnerabilities": 3
  }
}
```
### HTML отчет
Генерирует красивую HTML страницу с таблицей уязвимостей, сортировкой и фильтрацией.
